# Interface du gacha

Cette interface permet de communiquer avec le service des cartes
et d'obtenir aléatoirement des cartes exclusives.

Les chances sont les suivantes :
- 80% d'obtenir un ⭐.
- 15% d'obtenir un ⭐⭐.
- 4% d'obtenir un ⭐⭐⭐.
- 1% d'obtenir un ⭐⭐⭐⭐.
